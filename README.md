# MetaMask Ethereum Testnet Transfer
- 连接 **MetaMask** 钱包
- 获取当前账户地址和余额
- 监听账户变化
- 发起 **Sepolia 测试网** 转账

---

## 功能演示
- ✅ 点击按钮连接 MetaMask  
- ✅ 自动展示账户地址与 ETH 余额  
- ✅ 输入收款地址与金额后发起转账  
- ✅ 监听账户切换，自动更新界面   

---

## 环境要求
- Node.js >= 18
- 浏览器已安装 [MetaMask](https://metamask.io/)  
- Sepolia 测试网 ETH (可通过 [Goole Cloud Web3](https://cloud.google.com/application/web3/faucet/ethereum/sepolia) 领取)  

---

## 安装步骤

```bash
# 安装依赖
yarn install

# 启动
yarn dev

```
